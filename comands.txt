#1º passo: Gerar incidentes para análise#
No bash: python generate_incidents.py

#2º passo: Carrega a build do docker#
Abrir outro terminal: docker compose up --build

#3º passo: Carrega os incidentes criados#
No bash: docker compose exec api python -m src.db.load_csv

#4º passo: Abrir a página com os dados carregados#
No terminal do docker, ir na opção portas, na linha da "porta 8000", ir na coluna 
"Endereço Encaminhado e clicar na opção "Abrir no navegador".
